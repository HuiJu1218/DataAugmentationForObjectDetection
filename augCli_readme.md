# Usage

```
    python3 augCli.py -p <dataset path> -op <result path> -flip <flip action>
```
Example:
> python3 augCli.py -p /home/hueiru/Desktop/test_python/sample_D -op /home/hueiru/Desktop/test_python/sample_D/Augmentation_NG -flip 1
